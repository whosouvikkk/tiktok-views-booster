# 🚀 JEET TOOL (CLI Order Tool)

A modern Python terminal-based tool with a clean UI for collecting simple orders and sending them to Discord via webhook.

> ⚠️ This project is for **demo / educational purposes only**

---

## ✨ Features

* 🎨 Clean CLI UI using `rich`
* 📦 Simple order flow (Username, Video Link, Amount)
* ⚡ Fast & minimal design

---

## 🛠️ Requirements

Make sure you have Python installed (Python 3.8+ recommended)

Install required libraries:

```bash
pip install rich requests validators
```

---

## ⚙️ Setup

1. Clone this repository:

```bash
git clone https://github.com/whosouvikkk/tiktok-views-booster
```

2. Navigate to the project folder:

```bash
cd jeet-tool
```



## ▶️ How to Run

Run the script using:

```bash
python main.py
```

*(Replace `main.py` with your file name if different)*

---

## 🧠 How It Works

1. Select service (`TikTok Views`)
2. Enter:

   * Username
   * Video Link 
   * Amount 
3. Tool validates inputs
4. Places Order
5. Displays success message

---


## 📢 Output Example
